# Changelog

## 1.0.0

- Initial release
- Offline DeepSeek token counting using `@huggingface/tokenizers`
- Support for DeepSeek-V3, DeepSeek-R1, DeepSeek-Coder V2, DeepSeek-V2
- Status bar display with live token count and cost
- Model selection via command palette
- Click status bar for detailed breakdown
- Configurable model prices via VS Code settings
- Input/output token split with configurable ratio
- Built on official DeepSeek tokenizer from HuggingFace (100k vocab, BPE)
